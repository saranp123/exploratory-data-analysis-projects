# British Airways Review Dashboard

## Overview

This is an interactive **Tableau Public dashboard** that shows **customer reviews of British Airways**. You can explore ratings, trends, and feedback across different traveler types, service aspects, and regions.

## Dashboard Link

[View Dashboard](https://public.tableau.com/app/profile/saiganesh.pothakamuri/viz/BritishAirwaysReview_17682532135740/Dashboard)

## Features

* **Overall Ratings** – See average ratings and counts.
* **Trends Over Time** – Track changes in ratings by month or year.
* **Filters** – Select traveler type, seat class, or region.
* **Service Ratings** – Explore ratings for comfort, crew, food, value, etc.
* **Interactive Charts** – Hover and click to get more details.

## Data

* Customer reviews and ratings for British Airways
* Includes traveler type, seat class, service ratings, and region

## How to Use

1. Use filters to select specific travelers or regions.
2. Explore charts to see trends and comparisons.
3. Hover over data points for details.

## Tools

* Built with **Tableau Public**
