## Bike Buyers Analysis Dashboard

### Overview

This project analyzes customer demographic and lifestyle data to understand purchasing behavior for bikes. The goal is to identify trends and factors that influence whether a customer is likely to purchase a bike.

### Project Files

* Excel dataset containing customer demographics and purchase history
* Excel workbook with analysis and visualizations (if applicable)

### Features

* **Purchase Analysis** – Compare customers who purchased bikes vs those who did not
* **Demographic Breakdown** – Analyze age, gender, marital status, income, and education
* **Lifestyle Factors** – Explore commute distance, number of cars, and home ownership
* **Regional Insights** – Compare purchasing behavior across regions
* **Interactive Filtering (Excel)** – Slice data by key attributes to spot trends

### Data

* Customer demographic data including age, gender, income, education, and occupation
* Lifestyle attributes such as commute distance, car ownership, and home ownership
* Target variable indicating whether a customer purchased a bike

### How to Use

* Review summary tables and charts to understand key trends
* Filter by demographics (age, gender, income, region) to compare bike buyers vs non-buyers
* Use insights to infer which customer profiles are most likely to purchase a bike

### Tools

* Built with Microsoft Excel
* Uses Excel tables, pivot tables, and charts for analysis
